## Non-Goals

- No external databases, APIs, or paid services
- No user authentication beyond GitHub
- No real-time collaboration features (GitHub Issues handles this)
- No mobile-native app (dashboard is responsive web)
- No integration with project management tools (Jira, Linear, etc.)

## Technical Constraints

- **Zero dependencies**: no npm, no build step, no Docker
- **GitHub-native only**: Issues, Actions, Pages, Models API
- **Single-file dashboard**: all HTML, CSS, and JS in one file
- **All Actions pinned to SHA**: for supply-chain security
- **All `${{ }}` expressions in `env:` blocks**: prevents injection attacks
